# --- Hardcoded paths ---
# Tip: hardcoded usernames in paths leak via TWO patterns. The path WITH a
# trailing slash matches when something follows (/Users/foo/bin), but a
# bare path with no trailing slash (<string>/Users/foo</string>) needs its
# own rule. Most-specific (with-slash) FIRST so the broader rule does not
# pre-match. If you have multiple machines with different home dir names
# (e.g. personal `kpachhai` vs work `kiranpachhai`), list each one.
